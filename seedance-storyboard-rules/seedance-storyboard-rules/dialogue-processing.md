## 二十三、台词处理规则

### 核心铁律

**台词要求一字不增、一字不减、一字不改。**

剧本台词是神圣不可侵犯的，所有处理只能围绕原台词进行语气与重音设计，不能改动原文。

### 1. 情绪语气色彩匹配

根据情绪匹配以下四个维度的语气色彩：

#### (1) 声音节奏类

- **语速快慢**：紧张/激动 → 快；悲伤/沉重 → 慢；冷静/压迫 → 匀速。
- **停顿（气口）**：在逻辑重音前、情绪转折点、潜台词涌现处设置气口。

#### (2) 音强力度

- **音量强弱**：愤怒/爆发 → 强；委屈/隐忍 → 弱；威胁/压迫 → 低沉有力。

#### (3) 语调走向

| 语调类型 | 适用情绪 | 效果 |
|----------|----------|------|
| 升调 | 疑惑、反问、惊讶 | 悬念感、不确定 |
| 降调 | 肯定、愤怒、绝望 | 落定感、压迫感 |
| 平调 | 冷漠、麻木、镇定 | 疏离感、压制感 |
| 曲折调 | 讽刺、纠结、阴阳怪气 | 情绪复杂、言外之意 |

#### (4) 情绪语气色彩

与第二十二节情绪浓度值体系匹配，台词语气必须标注对应的情绪色彩。

**示例**：

```
台词："以后离我远点。"
情绪语气色彩：降调 + 低沉有力 + 愠怒（42%）混合鄙夷（44%）
```

### 2. 重音处理

根据剧本规定情境，分析抓取台词中表达事件和情绪的**最关键的字词或最不可缺少的字词**作为重音。

- 重音标注格式与第十九节一致：在台词后用括号标注。
- 重音必须基于规定情境分析，不能凭感觉。
- 一句台词最多标注 1-2 个重音，避免泛滥。

**示例**：

```
台词："你真以为花钱有用？"
重音：（重音落在"真"，强调质疑）
```

### 台词处理输出格式

每句重要台词必须输出：

```text
台词："原文"
情绪语气色彩：语速 + 音强 + 语调 + 情绪色彩（浓度值 %）
重音：（重音落在"X"，原因）
潜台词：（如有，参考第八节）
```

### 核心原则

**台词是骨架，语气是血肉，重音是关节，潜台词是灵魂。**
