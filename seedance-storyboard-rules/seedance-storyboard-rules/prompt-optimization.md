## 十九、提示词优化规则

在优化 Seedance 2.0 提示词时，必须遵守以下三条核心原则。

### 1. 保留情绪表达

优化提示词时，不能为了精简 token 或追求简洁而丢失情绪描述。

- 情绪表达是 Seedance 2.0 表演的核心依据。
- 微动作清单（参考第七节）中的情绪动作必须保留。
- 双层表演（参考第九节）的外层与内层情绪必须同时保留。
- 表情三段式（起始 → 变化 → 最终）必须完整保留。

**禁止**：将"眼皮压低、瞳孔收缩、单侧嘴角上扬"简化为"冷笑"。

### 2. 保留逻辑重音

台词中的逻辑重音（需要强调的字词）必须保留，并单独在台词后用括号标注。

**标注格式**：

```
台词："以后离我远点。"
逻辑重音：（重音落在"远"）
```

**示例**：

| 台词 | 逻辑重音标注 |
|------|--------------|
| "台阶？" | （重音落在"台阶"，反问语气） |
| "以后离我远点。" | （重音落在"远"） |
| "你真以为花钱有用？" | （重音落在"真"） |

逻辑重音决定了台词的语气走向，影响微表情设计与声音表演，优化时不可省略。

### 3. 潜台词与剧本台词区分

当出现潜台词时，必须与剧本台词明确区分，不能混为一谈。

**区分格式**：

- **剧本台词**：角色实际说出口的话，用引号标注。
- **潜台词**：角色心里真正想表达的话，用"潜台词："明确标注，与台词分行。

**示例**：

```
台词："台阶？"
潜台词：真搞笑，大家快来看，这女的到底在说什么莫名其妙的话。
```

**禁止**：

- 将潜台词混入台词中。
- 将台词与潜台词写在同一行。
- 优化时删除潜台词标注。

### 4. 提示词与剧本顺序逐字咬合

分镜提示词必须按剧本顺序逐字咬合，不能跳序、不能合并、不能遗漏。

- 情绪过渡要拆分细腻，不能一笔带过。
- 提示词颗粒度也要细腻，每个情绪节点都要有对应提示词。
- 剧本中每个动作、每句台词、每个情绪转折都必须在提示词中有对应体现。

**禁止**：

- 跳过剧本中的某个情绪节点。
- 将多个镜头的情绪合并为一段提示词。
- 提示词与剧本顺序不一致。

### 5. 台词一字不改

**台词要求一字不增、一字不减、一字不改。**

- 提示词中引用的台词必须与剧本原文完全一致。
- 不能为了提示词通顺而修改台词用词。
- 不能为了情绪表达而增删台词字词。
- 台词的标点符号也必须保留原样。

### 核心原则

**优化提示词是精简冗余，不是精简情绪与逻辑。**
