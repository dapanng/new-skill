## 二十一、核心分析公理与人物关系情绪

### 三大核心公理

#### 公理一：情境催生一切

所有人物行动、情绪、台词、肢体反应，**全部由规定情境催生**。

- 离开规定情境分析人物情绪 = 无源之水。
- 任何情绪必须能追溯到具体的规定情境条件。
- 规定情境改变，人物情绪必须随之变化。

#### 公理二：人物关系决定情绪表达（重中之重）

**人物关系，对不同人物会有不同的情绪表达。这是重中之重，对人物分析情绪很重要。**

同一人物面对不同对象，情绪表达方式完全不同：

| 对象关系 | 情绪表达特征 |
|----------|--------------|
| 面对爱人 | 柔化、克制、患得患失 |
| 面对仇人 | 隐忍或爆发、攻击性 |
| 面对长辈 | 压抑、恭敬或反抗 |
| 面对下属 | 强势、疏离或亲和 |
| 面对陌生人 | 防备、客套或冷漠 |
| 面对情敌 | 警惕、嫉妒或鄙夷 |

**分析要求**：

每场戏必须先梳理人物关系矩阵，再分析每个角色在该关系下的情绪表达方式。

#### 公理三：性格分析驱动情绪浓度

根据规定情境进行人物性格分析，根据每个人物关系以及事件，**超级详细地分析每场戏的人物情绪**，要拆分得非常非常细腻，以及情绪浓度值分析。

情绪浓度值按百分比计算，具体情绪与浓度值匹配参见第二十二节。

### 人物关系分析输出格式

```text
【人物关系情绪分析】

角色 A → 角色 B：
- 关系定位：
- 当下关系状态：
- A 对 B 的表层情绪：（浓度值 %）
- A 对 B 的深层情绪：（浓度值 %）
- 表达方式：（外放/隐忍/伪装/矛盾）

角色 B → 角色 A：
- 关系定位：
- 当下关系状态：
- B 对 A 的表层情绪：（浓度值 %）
- B 对 A 的深层情绪：（浓度值 %）
- 表达方式：（外放/隐忍/伪装/矛盾）
```

### 情绪拆分要求

- 每场戏的情绪分析必须**超级详细**，拆分到非常细腻的颗粒度。
- 不能只写"愤怒"，要拆为：愠怒（42%）→ 强忍怒火（32%）→ 暴怒（94%）的浓度变化曲线。
- 情绪浓度值必须标注百分比。
- 情绪变化必须有起始 → 发展 → 结果的完整过程。

### 核心原则

**人物关系是情绪表达的变量，规定情境是情绪催生的常量。**
