## 十六、声音设计规则

### 默认无 BGM、无字幕

**硬性要求**：

- **禁止 BGM**（背景音乐）
- **禁止字幕**（Screen Text / Subtitle）
- **需要音效**（Sound Effect / Foley）

声音分为：

| 类型 | 说明 |
|------|------|
| 对白 | 角色台词 |
| OS 内心独白 | 必须明确：无口型 |
| 环境声 | 场景背景音 |
| 动作音效 | 动作产生的声音 |
| 群杂 | 群像背景人声 |
| 主观听觉 | 角色心理听觉 |

### 音效设计要求

每个分镜必须明确标注所需音效，音效是叙事的重要组成部分：

- **环境音**：风声、人群嘈杂、交通声等场景底噪。
- **动作音效**：脚步声、门开关、物品掉落、拍桌等。
- **Foley**：衣物摩擦、手指敲击、呼吸声等拟音。
- **主观听觉**：耳鸣、心跳、低通滤波等心理音效（参考下文）。

### 主观心理变化技法

可以使用：

- 耳鸣
- 心跳
- 低通滤波
- 声音远化
- 环境声骤降

**但不能滥用。**

### L-CUT / J-CUT

用于连接连续台词：

```
上一镜人物OS开始
→ 画面切换
→ 声音不断
→ 下一镜完成句子
```
