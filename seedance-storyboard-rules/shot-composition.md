## 三、画面运作规则

每一个镜头都要明确四种焦点角色：

| 焦点类型 | 说明 |
|----------|------|
| 绝对焦点 | 画面视觉中心 |
| 动作焦点 | 当前正在执行动作的人物 |
| 反应焦点 | 承接动作并产生情绪反应的人物 |
| 空间锚点 | 仅用于标定空间位置，不抢戏 |

### 单一视觉核心原则

画面不能所有人物同时抢戏。**一个镜头只允许有一个明确视觉核心。**

### 动作交接焦点

**示例**：陈洛转身离开时

- 陈洛 = 动作焦点
- 夏清月 = 反应焦点

镜头可以先跟陈洛半步，再停住，把陈洛放出焦点，让夏清月成为最终视觉中心。

### 台词拍摄原则

人物说话时，不一定始终拍说话者。可以利用：

```
台词人物 → 被评价对象 → 反应人物
```

完成电影级叙事。
