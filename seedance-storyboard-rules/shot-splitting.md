## 二、Seedance 2.0 单个视频的镜头拆分原则

每个分镜必须考虑 Seedance 2.0 单段视频的可控时长。

### 核心原则

**一个视频只承担一个主要叙事动作或一个主要情绪变化。**

不要在一个镜头里同时塞入：

- 复杂走位 + 多次转身 + 多段对白 + 多次情绪反转 + 多个特写

### 复杂段落拆分逻辑

复杂段落应拆成：

```
动作建立 → 物件插入镜头 → 人物反应 → 台词落点 → 情绪结果
```

**示例**：

```
双人对峙
→ 项链特写
→ 手腕饰品特写
→ 高跟鞋特写
→ 回双人镜头完成台词反杀
```

这样比一个镜头要求模型同时完成所有动作更稳定。

### 时长建议

| 时长 | 适用场景 |
|------|----------|
| 2—3 秒 | 插入镜头、物件特写、眼神反应、单一微表情 |
| 3—5 秒 | 一句完整台词、人物单次动作、简单运镜 |
| 5—7 秒 | 双人关系变化、环绕、起身、转身离开等完整动作 |

### A/B/C/D 镜拆分法

如一个镜头动作超过模型稳定表现范围，必须主动拆分为 A / B / C / D 镜。
