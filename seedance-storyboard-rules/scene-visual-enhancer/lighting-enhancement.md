## 四、光影优化规则

光影增强只作用于光层，不触碰布局层。每条规则均以正向描述表达。

### 光源类型

| 光源类型 | 正向描述方式 | 适用情绪 |
|----------|--------------|----------|
| 主光 | 「侧上方主光塑造立体轮廓」 | 坚定、专注、权威 |
| 辅光 | 「弱辅光建立暗部反光层次」 | 内敛、沉思、温和 |
| 轮廓光 | 「后方轮廓光勾勒人物边缘」 | 分离感、神圣、戏剧 |
| 环境光 | 「环境反射光填充整体层次」 | 自然、生活感、沉浸 |
| 眼神光 | 「眼球反射点状眼神光」 | 灵动、真诚、聚焦 |

### 明暗关系与暗部分寸感

从美术角度建立明确的明暗关系，暗部保持分寸感。明暗关系是画面体积与空间感的根基，暗部处理决定画面的「厚重感」与「呼吸感」。

- **明确的明暗分界**：画面建立清晰的光面、明暗交界线、暗面三层结构，明暗交界线呈现可辨认的转折过渡，光暗关系一眼可读。
- **明暗交界线最暗**：明暗交界线（core shadow）呈现暗部最暗的色阶，反光区略亮于交界线，构成「亮面 → 明暗交界线（最暗）→ 反光区（略亮）→ 投影」的美术明暗结构。
- **暗部分寸感**：暗部保持克制的细节密度，呈现少量关键质感线索（如衣褶走向、发丝方向），而非完整细节铺陈；细节服务于体积认知，不抢夺视觉焦点。
- **暗部非纯黑**：暗部保留低明度的色相与层次，呈现深灰、深褐、深蓝等有色暗调，而非 RGB(0,0,0) 的死黑；最深处保留与环境光对应的微量层次。
- **暗部色彩倾向**：暗部细节呈现与环境光对应的色温倾向——冷光环境下暗部偏青蓝、反光偏暖；暖光环境下暗部偏深褐橙、反光偏冷，呈现「冷光源 + 暖反光」或「暖光源 + 冷反光」的补色关系。
- **个别反光源**：暗部允许出现个别点状或线状反光（bounce light），如来自地面/桌面/衣物的环境反射光在暗部边缘形成柔和反光带；反光强度低于主光，位置服务于体积转折，不破坏明暗关系。
- **明暗优先于细节**：当暗部细节与明暗关系冲突时，优先保全明暗关系的整体性，牺牲细节还原；细节是明暗关系的从属，不喧宾夺主。

### 光位与角度

| 光位 | 画面效果（正向） | 情绪指向 |
|------|------------------|----------|
| 顺光 | 呈现整体均匀照度与色彩饱和 | 明朗、平实、记录感 |
| 侧光 | 塑造立体轮廓与质感纹理 | 坚毅、专注、雕塑感 |
| 侧逆光 | 勾勒边缘轮廓并保留正面暗部层次 | 戏剧、神秘、分离感 |
| 逆光 | 形成边缘光环与体积光透射 | 神圣、怀念、梦幻 |
| 顶光 | 在头顶与眉骨投下垂直阴影，强化骨骼结构 | 严肃、压抑、审判感 |
| 底光 | 自下而上照亮面部，呈现阴影翻转 | 诡异、紧张、戏剧张力 |

### 时间与色温

| 时段 | 光色与色温（正向） | 适用场景 |
|------|--------------------|----------|
| 清晨 | 4500K 偏冷蓝调低角度侧光，呈现清透空气感与晨雾层次 | 室内外晨戏、醒来、出发 |
| 正午 | 5500K 中性白顶光，呈现高照度与短阴影 | 室外街景、办公、对话 |
| 黄金时刻 | 3200K 暖橙调低角度侧光，呈现温暖怀旧情绪 | 室外告别、回忆、浪漫 |
| 蓝调时刻 | 7500K 冷蓝调漫射光，呈现静谧忧郁层次 | 都市夜景、过渡、独处 |
| 夜晚 | 2800K 暖色点光源与冷蓝环境光对比，呈现霓虹反射与烛火暖光 | 室内夜戏、雨夜、酒吧 |

### 材质质感

- 皮肤次表面散射：呈现耳垂/鼻翼/指尖的透红血色。
- 眼球湿润高光：眼球表面呈现湿润镜面反射与点状眼神光。
- 金属反射：金属表面呈现锐利高光与环境反射层次。
- 织物纹理：织物呈现纤维方向与柔软漫反射。
- 湿润路面反射：路面呈现霓虹/车灯的镜面反射与色光晕开。
- 木质纹理：木材呈现年轮方向与哑光漫反射。
- 玻璃通透：玻璃呈现边缘折射与表面环境反射叠加。

### 脸部重点

- 眼部高光：眼球呈现点状或环状眼神光。
- 鼻梁颧骨立体感：侧光在鼻梁与颧骨塑造明暗过渡，明暗交界线落在骨点转折处。
- 下颌线阴影：下颌呈现柔和过渡阴影以分离颈部，阴影保持有色暗调而非纯黑。
- 自然皮肤高光：T 区与颧骨呈现自然微量油光。
- 克制的暗部细节：暗部保留少量关键质感线索（如发丝方向、衣褶走向），呈现色彩倾向与个别反光，而非完整细节铺陈。
- 嘴唇湿润反光：嘴唇呈现湿润点状高光与轮廓过渡。

### 禁止项的正面转化

| 原禁止项（仅参考） | 正向描述（应呈现） |
|--------------------|-------------------|
| 过曝 | 保留高光与暗部的合理动态范围，高光区呈现细节而非纯白 |
| 塑料皮肤 | 呈现自然皮肤纹理、毛孔质感与次表面散射血色 |
| 全脸均匀打亮 | 通过侧光塑造立体轮廓，保留明暗过渡 |
| flat lighting | 以侧光/侧逆光建立方向性光位，呈现明暗层次与立体感 |
| 死黑阴影 | 暗部呈现有色暗调（深灰/深褐/深蓝），保留与环境光对应的色温倾向、微量层次与个别反光，而非纯黑 |
| 暗部细节堆砌 | 暗部保持克制的细节密度，呈现少量关键质感线索而非完整铺陈，明暗关系优先于细节还原 |
| 暗部无色彩 | 暗部呈现与环境光补色的色温倾向（冷光+暖反光 / 暖光+冷反光），暗部细节带色相而非灰阶脱水 |
| 过饱和 | 呈现符合时段色温的合理饱和度，肤色与场景色协调 |
| 廉价高光 | 高光呈现材质属性一致的反射与衰减过渡 |
