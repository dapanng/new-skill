## 使用工作流

### 何时调用此 Skill

调用本 Skill 的典型场景：

- 用户要求优化场景提示词的光影或机位
- 用户要求在不改布局前提下增强画面
- 用户要求室内或室外场景的视觉增强策略
- 用户要求把原提示词转为正向视觉增强
- 用户使用 Seedance / HyperFrames / Remotion 等工具需要正向视觉提示词增强
- 用户要求审查既有增强结果是否符合「布局保真 + 正向比例」

### 如何套用优化模板

标准五步流程：

1. **第一步：锁定原提示词**（参考 `layout-preservation.md`）
   - 按五类要素（人物/物件/空间/动作/时间天气）提取布局锚点
   - 原文未指定的标注「未指定」

2. **第二步：判定室内/室外并选择对应策略**（参考 `indoor-strategy.md` / `outdoor-strategy.md`）
   - 判定场景类型与子类型
   - 选择对应的主光源/机位/氛围策略

3. **第三步：光影增强**（参考 `lighting-enhancement.md`）
   - 选择光源类型、光位、时间色温
   - 按 `positive-visual-style.md` 的句式模板生成正向描述

4. **第四步：机位与氛围增强**（参考 `camera-enhancement.md`、`atmosphere-enhancement.md`）
   - 选择机位、焦距、运镜、构图
   - 选择色调、空气感、体积光、反射
   - 每条增强与布局锚点对照，冲突即放弃

5. **第五步：填写 `templates/enhancement.template.md` 并完成布局保真校验**
   - 逐项填写模板字段
   - 完成「布局保真声明」与「负向提示词（仅校验用）」

### 如何校验

按以下清单逐项检查：

| 校验维度 | 检查内容 | 对应章节 |
|----------|----------|----------|
| 布局保真 | 人物/物件/空间/动作/时间天气是否与原提示词一致 | 第二节 |
| 正向比例 | 正向增强区是否全为肯定句，无否定句承担增强 | 第三节 |
| 室内外匹配 | 是否按场景类型选择对应室内/室外策略 | 第七/第八节 |
| 光影服务情绪 | 光影是否服务场景已有情绪，是否炫技 | 第四节、第一节 |
| 机位不矛盾 | 机位是否导致人物/物件相对位置矛盾 | 第五节 |
| 氛围不抢焦 | 氛围是否遮挡关键人物面部或关键物件 | 第六节 |
| 模板完整 | 是否完整填写模板所有字段，含布局保真声明 | `templates/enhancement.template.md` |

### 输出规范

- 所有优化输出必须使用 `templates/enhancement.template.md` 的固定模板。
- 原提示词必须逐字保留在「原提示词」字段。
- 正向增强区（光影/摄像机/氛围）不得出现否定句承担增强职责。
- 末尾必须含「布局保真声明」，确认布局一致与正向比例。
- 「负向提示词（仅校验用）」字段可选填，不计入正向视觉增强主体。
- 连续多场景优化时，每个场景独立填写一份模板，不得跨场景混用锚点。
