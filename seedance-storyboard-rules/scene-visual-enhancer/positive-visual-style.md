## 三、正面视觉表达规则

正面视觉是本 Skill 的表达底线：所有增强只描述「画面是什么样」，不描述「画面不要什么样」。

### 正向描述原则

- 所有增强以肯定句式表达，如「画面呈现 X」「光线从 Y 方向打入形成 Z」「镜头位于 A 位置呈现 B 透视」。
- 描述聚焦于可观察的视觉特征：光源、光位、光比、色温、焦距、景深、机位、构图、空气感、色调。
- 每条增强应可被画师/渲染器直接实现，无需推理「不要什么」。
- 优先使用具体可量化的视觉参数（如焦距毫米数、色温 K 值、光位角度），避免主观情绪词替代视觉描述。
- 一条增强只承载一个视觉层信息，光源、机位、氛围分别成句，不混排。
- 情绪指向作为描述的收束词出现，不作为描述主体，如「侧光塑造立体轮廓，呈现专注情绪」而非「专注的侧光」。

### 禁用负向承担增强

禁止用「不要过曝」「不能塑料皮肤」「避免 flat lighting」「禁止全脸均匀打亮」等否定句作为增强内容。此类约束只能出现在模板末尾的「负向提示词（仅校验用）」字段，且不参与本次「正向视觉增强」主体。

判定规则：凡含「不要 / 不能 / 避免 / 禁止 / 不可 / 勿 / 不得」等否定词的句子，均不得作为增强主体输出；如需保留约束语义，必须改写为对应的正向画面描述，例如「不要过曝」应改写为「保留高光区细节与合理动态范围」。

### 句式模板

```
<光源> 从 <方向> 以 <角度> 打入，在 <主体> 上形成 <光效>，呈现 <情绪>。
镜头位于 <机位高度> 以 <焦距> 拍摄 <主体>，<景深> 虚化 <前后景>，呈现 <透视感>。
画面以 <色调> 为主调，<空气感元素> 在 <光位> 中呈现 <层次>，整体 <情绪>。
<主体> 处于 <光位>，<材质> 呈现 <质感细节>，明暗交界线落在 <转折处>，暗部保持 <有色暗调> 与 <个别反光>，高光保留 <细节>。
```
