## 一、核心原则

整套场景视觉增强始终遵守以下不可妥协原则：

1. **布局绝对不变**：原提示词中的人物站位、物件位置、空间结构、动作意图、台词落点一律保留，优化只作用于光影、摄像机、氛围、材质质感层。
2. **只加不减**：只追加正向视觉描述，不删除原提示词的任何元素，不引入负向提示词条目。
3. **正面视觉**：所有增强必须以正向视觉描述表达（描述「画面是什么样」），禁止用「不要/不能/避免」等否定句式承担增强职责；否定句仅允许出现在独立的「布局保真声明」中用于校验。
4. **光影服务情绪**：光影与机位必须服务场景已有情绪，不为炫技而新增与原场景无关的戏剧张力。
5. **室内外分治**：室内与室外采用不同的光源、机位、氛围策略，不可混用通用模板。
6. **因果可追溯**：每条增强描述必须能追溯到原提示词的某个要素（人物/物件/空间/时间），不得无中生有。

### 因果链路

```
原提示词锁定 → 布局锚点固化 → 光影增强 → 机位增强 → 氛围增强 → 布局保真校验 → 输出
```
