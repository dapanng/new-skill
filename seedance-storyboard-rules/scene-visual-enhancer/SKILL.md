# 场景视觉提示词优化 Skill

## 概述

本 Skill 用于在 AI 视频/图像生成流程中，对已有的「场景原提示词」做光影、摄像机、氛围的正向视觉增强。其核心约束是「只加不减、只正不负、不改布局、室内外分治」：即在锁定原提示词所描述的场景内部布局之后，仅在其上叠加正向的光影、机位与氛围描述，不删除、不否定、不重排原有要素，并对室内与室外场景采用不同的光源、机位与氛围策略。

本 Skill 产出的正向视觉增强提示词为规则化的纯文本，可由用户手动粘贴到下游工具的提示词字段中，例如 Seedance 分镜的镜头提示词、HyperFrames HTML 合成的视觉描述字段、Remotion React 视频的画面描述字段等。本 Skill 自身不调用这些下游工具的代码运行能力，仅产出可被人工迁移使用的提示词文本。

**核心理念**：

- 布局先锁定，光影再增强。
- 只加不减，只正不负。
- 光影与机位服务已有情绪，不为炫技新增戏剧张力。
- 室内与室外采用不同光源、机位、氛围策略。
- 每条增强必须可追溯到原提示词的某个要素。
- 增强结果必须可校验：布局保真 + 正向比例。

**最终形成的因果链路**：

```
原提示词锁定 → 布局锚点固化 → 光影增强 → 机位增强 → 氛围增强 → 布局保真校验 → 输出
```

## 加载时必读

按以下顺序阅读本 Skill 的各主题文件：

1. **`core-principles.md`** — 一、核心原则（布局绝对不变、只加不减、正面视觉、光影服务情绪、室内外分治、因果可追溯）。
2. **`layout-preservation.md`** — 二、场景布局保持规则（布局要素清单、锁定流程、禁止项、冲突判定）。
3. **`positive-visual-style.md`** — 三、正面视觉表达规则（正向描述原则、禁用负向承担增强、正向词汇库、句式模板）。
4. **`lighting-enhancement.md`** — 四、光影优化规则（光源类型、光位角度、时间色温、材质质感、脸部重点）。
5. **`camera-enhancement.md`** — 五、摄像机优化规则（机位高度角度、焦距景深、运镜、构图、机位与布局关系）。
6. **`atmosphere-enhancement.md`** — 六、氛围与色调优化规则（色调情绪映射、空气感、丁达尔与体积光、环境反射）。
7. **`indoor-strategy.md`** — 七、室内场景专属策略（住宅、办公商业、公共、特殊室内四类）。
8. **`outdoor-strategy.md`** — 八、室外场景专属策略（城市街道、自然景观、开阔场地、特殊室外四类）。
9. **`workflow.md`** — 使用工作流（何时调用、如何套用优化模板、如何校验、输出规范）。
10. **`templates/enhancement.template.md`** — 固定优化输出模板（每镜统一格式）。

在用户开始优化场景提示词之前，先确认其原提示词已提供，再按 `workflow.md` 的工作流套用模板。
