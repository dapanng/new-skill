## 使用工作流

### 何时调用此 Skill

调用本 Skill 的典型场景：

- 用户要求创作微短剧分镜（无论是否基于 Seedance 2.0）
- 用户要求拆分镜头、规划镜头序列
- 用户要求编写 Seedance 2.0 / 类似 AI 视频模型的提示词
- 用户要求设计电影级 CG 分镜
- 用户要求套用或修改固定分镜模板
- 用户讨论空间连续性、景别衔接、运镜、双层表演等具体技法
- 用户要求审查既有分镜是否符合规则

### 如何套用分镜模板生成新分镜

1. **第一步：规定情境分析（新增·首要步骤）**
   - 完成外部规定情境分析（参考第二十节）
   - 完成内部规定情境分析（参考第二十节）
   - 梳理人物关系矩阵与情绪表达方式（参考第二十一节）
   - 这是所有后续步骤的前提，不完成不能进入下一步

2. **第二步：情绪浓度值分析（新增）**
   - 根据规定情境与人物关系，分析每场戏的情绪（参考第二十一节）
   - 为每个情绪节点匹配浓度值百分比（参考第二十二节）
   - 形成情绪浓度变化曲线（起始 → 发展 → 结果）
   - 提取对应的 CDS2.0 微表情提示词

3. **第三步：空间与关系建立**
   - 明确场景空间结构（参考第十三节）
   - 明确人物站位、距离、朝向（参考第四节）
   - 锁定光影与时间（参考第十四节）

4. **第四步：镜头拆分**
   - 按"一视频一动作/一情绪"原则拆分（参考第二节）
   - 复杂段落按 A/B/C/D 镜拆分
   - 估算每镜时长（2-3s / 3-5s / 5-7s）

5. **第五步：分镜三层级设计（新增）**
   - 叙事层：保证剧情信息完整（参考第二十五节）
   - 情绪层：贴合情绪浓度值（参考第二十五节）
   - 镜头衔接层：流畅自然转场（参考第二十五节）

6. **第六步：填写分镜模板**
   - 使用第十七节固定分镜模板
   - 逐项填写：镜号、时长、人物空间、运镜景别、画面动作、细微动作、声场、台词、正/负向提示词

7. **第七步：台词处理（新增）**
   - 台词一字不增、不减、不改（参考第二十三节）
   - 匹配情绪语气色彩：语速 + 音强 + 语调 + 情绪色彩（参考第二十三节）
   - 标注重音：在台词后括号标注（参考第十九、二十三节）
   - 确定潜台词：与台词分行区分（参考第八节）

8. **第八步：表演与肢体动作设计**
   - 将情绪拆解为微动作清单（参考第七节）
   - 匹配情绪浓度值的微表情提示词（参考第二十二节）
   - 设计肢体动作：基于情绪浓度与物件绑定（参考第二十四节）
   - 双层人物按"表情在演，眼睛泄密"设计（参考第九节）

9. **第九步：填写正向/负向提示词**
   - 正向提示词：以第一节默认提示词为基础 + 本镜具体描述 + 情绪浓度微表情
   - 负向提示词：列出禁止项（文字、字幕、水印、BGM、过曝等）
   - 提示词按剧本顺序逐字咬合，情绪拆分细腻（参考第十九节）

### 如何校验分镜是否符合规则

按以下清单逐项检查：

| 校验维度 | 检查内容 | 对应章节 |
|----------|----------|----------|
| 空间连续性 | 上一镜与下一镜的站位、距离、朝向是否一致 | 第四节 |
| 焦点统一性 | 每镜是否只有一个视觉核心 | 第三节 |
| 时长合理性 | 时长是否匹配动作复杂度 | 第二节 |
| 景别衔接 | 是否避免连续近景，是否使用视线/动作匹配 | 第五节 |
| 运镜服务性 | 运镜是否服务情绪，是否为炫技 | 第六节 |
| 表演具体性 | 情绪是否拆为微动作，是否使用情绪词 | 第七节 |
| 潜台词设计 | 重要台词是否标注潜台词 | 第八节 |
| 双层表演 | 绿茶/虚伪角色是否同时存在两层情绪 | 第九节 |
| 群像多样 | 群演反应是否多样，群杂是否层层叠加 | 第十节 |
| 手机处理 | 手机是否作为环境元素，是否遮挡主体 | 第十一节 |
| 物件叙事 | 物件是否绑定动作，是否承担剧情功能 | 第十二节 |
| 场景稳定 | 场景元素是否无故消失 | 第十三节 |
| 光影统一 | 光影是否与时间天气统一 | 第十四节 |
| 蒙太奇拆分 | 心理蒙太奇是否拆为快切 | 第十五节 |
| 声音设计 | 是否无 BGM，OS 是否无口型 | 第十六节 |
| 模板完整 | 是否完整填写固定分镜模板所有字段 | 第十七节 |
| 提示词优化 | 情绪表达是否保留、逻辑重音是否标注、潜台词是否与台词分行区分、提示词是否按剧本顺序咬合、台词是否一字不改 | 第十九节 |
| 规定情境分析 | 是否完成外部 + 内部规定情境分析 | 第二十节 |
| 人物关系情绪 | 是否梳理人物关系矩阵，是否分析不同对象的不同情绪表达 | 第二十一节 |
| 情绪浓度值 | 是否标注浓度百分比，是否形成浓度曲线，是否匹配微表情提示词 | 第二十二节 |
| 台词处理 | 台词是否一字不改，是否标注情绪语气色彩，是否标注重音 | 第二十三节 |
| 肢体动作 | 肢体动作是否匹配情绪浓度，是否与物件绑定 | 第二十四节 |
| 分镜三层级 | 叙事层是否完整、情绪层是否贴合、衔接层是否流畅 | 第二十五节 |

### 输出规范

- 所有分镜输出必须使用第十七节的固定模板
- 正向提示词必须以第一节默认统一提示词开头
- 负向提示词必须包含：文字、字幕、水印、BGM、过曝、塑料皮肤
- 连续分镜之间必须明确标注空间关系继承
- 复杂段落必须显式标注 A/B/C/D 镜拆分
- 台词后必须用括号单独标注逻辑重音，如：（重音落在"远"）
- 潜台词必须与剧本台词分行区分，用"潜台词："明确标注，禁止混入台词或同行书写
- 台词一字不增、不减、不改，标点符号保留原样
- 每句重要台词必须标注情绪语气色彩（语速 + 音强 + 语调 + 情绪色彩浓度值）
- 每个分镜必须标注所需音效（环境音 / 动作音效 / Foley / 主观听觉）
- 情绪必须标注浓度值百分比，并匹配第二十二节的微表情提示词
- 提示词必须按剧本顺序逐字咬合，情绪过渡拆分细腻
- 分镜设计必须依次满足叙事层 → 情绪层 → 镜头衔接层三层级
- 每场戏开始前必须先输出规定情境分析（外部 + 内部）
