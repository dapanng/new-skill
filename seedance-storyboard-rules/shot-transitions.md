## 五、电影级景别衔接规则

### 避免长期连续近景

禁止：近景 → 近景 → 近景 的连续组合。

### 景别递进链路

```
全景/中远景（建立空间）
→ 中景（建立人物关系）
→ 中近景（推进冲突）
→ 近景（进入情绪）
→ 特写（完成心理落点）
```

### 衔接原则

下一镜如果已经是近景，那么上一镜尽量使用：中景、中近景、过肩关系镜头。

### 衔接技法

- 视线匹配 Eyeline Match
- 动作匹配 Match on Action
- 焦点转移 Rack Focus
- L-CUT
- J-CUT

**示例**：

```
人物A说完
→ 视线转向人物B
→ 停半拍
→ 切人物B近景
```

这样下一镜近景是上一镜制造出来的"结果"，而不是机械换脸。
