# 固定分镜模板

> 此模板来自原 Skill 第十七节，使用时按需填写各项字段。

```text
【镜号】镜头标题 / 情绪功能

时长：
景别：
运镜：
角度：

画面动作：

下意识细微动作：

全系声场：
台词情绪：
台词：
逻辑重音：
环境音：

人物与空间：

正向提示词：

负向提示词：
```

## 字段填写说明

| 字段 | 填写要求 | 对应规则 |
|------|----------|----------|
| 镜号 | 唯一编号，如 S01-A / S01-B（复杂段落用 A/B/C/D 拆分） | 第二节 |
| 镜头标题 / 情绪功能 | 一句话概括本镜动作与情绪功能 | 第三节、第二十一节 |
| 时长 | 2-3s / 3-5s / 5-7s，按动作复杂度选择 | 第二节 |
| 景别 | 全景 / 中景 / 中近景 / 近景 / 特写，按情绪浓度选择 | 第五节、第二十五节 |
| 运镜 | 慢推 / 环绕 / 手持 / 静止 / 焦点转移 / 插入镜头 | 第六节 |
| 角度 | 平视 / 俯视 / 仰视 / 倾斜，服务叙事与情绪 | 第二十五节 |
| 画面动作 | 明确肢体动作 + 与物件的交互 | 第二十四节 |
| 下意识细微动作 | 微动作清单，匹配情绪浓度值（第二十二节微表情提示词） | 第七节、第二十二节 |
| 全系声场 | 对白 / OS / 动作音效 / 群杂 / 主观听觉（无 BGM、无字幕） | 第十六节 |
| 台词情绪 | 语速 + 音强 + 语调 + 情绪色彩（浓度值 %） | 第二十三节 |
| 台词 | 剧本原文，一字不增、不减、不改 | 第二十三节 |
| 逻辑重音 | 台词后括号标注，如（重音落在"远"） | 第十九节、第二十三节 |
| 环境音 | 场景环境音 / Foley / 主观听觉 | 第十六节 |
| 人物与空间 | 站位、距离、朝向、左右前后位置，与上一镜连续 | 第四节、第十三节 |
| 正向提示词 | 以第一节通用质量基底为基础 + 选定风格提示词 + 本镜具体描述 + 情绪浓度微表情 | 第一节、第十九节 |
| 负向提示词 | 文字、字幕、水印、BGM、过曝、塑料皮肤等禁止项 | 第一节、第十九节 |
