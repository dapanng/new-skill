## 十六、声音设计规则

### 默认无 BGM

声音分为：

| 类型 | 说明 |
|------|------|
| 对白 | 角色台词 |
| OS 内心独白 | 必须明确：无口型 |
| 环境声 | 场景背景音 |
| 动作音效 | 动作产生的声音 |
| 群杂 | 群像背景人声 |
| 主观听觉 | 角色心理听觉 |

### 主观心理变化技法

可以使用：

- 耳鸣
- 心跳
- 低通滤波
- 声音远化
- 环境声骤降

**但不能滥用。**

### L-CUT / J-CUT

用于连接连续台词：

```
上一镜人物OS开始
→ 画面切换
→ 声音不断
→ 下一镜完成句子
```
