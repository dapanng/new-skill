## 七、人物表演规则

### 禁止单一情绪词

人物情绪禁止只写：愤怒、开心、悲伤、冷笑。

必须拆成 Seedance 2.0 能表现的具体动作。

### 微动作清单

| 部位 | 微动作 |
|------|--------|
| 眼部 | 眼皮压低 |
| 眼部 | 瞳孔收缩 |
| 眼部 | 眼球短促侧移 |
| 眉部 | 单侧眉峰轻挑 |
| 嘴部 | 单侧嘴角上扬 |
| 嘴部 | 单侧嘴角下压 |
| 鼻部 | 鼻翼轻微收紧 |
| 喉部 | 喉结滚动 |
| 嘴部 | 嘴唇发紧 |
| 下颌 | 下颌放松或绷紧 |
| 呼吸 | 短促鼻息 |
| 肩部 | 肩膀僵硬 |
| 脚部 | 脚尖下意识移动 |
| 手部 | 手指蜷缩 |
| 手部 | 拇指摩擦食指 |
| 手部 | 偷偷掐掌心 |

### 表情三段式

表情必须有：

```
起始状态 → 变化过程 → 最终状态
```

**示例 1**：

```
震惊
→ 偷瞄四周
→ 意识到全场都在看
→ 社死感冲顶
```

**示例 2**：

```
柔弱伪装
→ 短暂空白
→ 慌乱泄露
→ 条件反射恢复伪装
```
