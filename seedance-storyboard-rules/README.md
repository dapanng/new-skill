# Seedance 2.0 微短剧分镜规则 Skill — 快速入门

Seedance 2.0 微短剧分镜创作的全套全局规则体系，共 18 节，覆盖视觉渲染、镜头拆分、空间连续性、景别衔接、运镜、表演、潜台词、群像、声音设计等全链路要素。

---

## 内含内容

- **`SKILL.md`** — 技能入口与索引导航（首先读取）。
- **18 个主题 `.md` 文件** — 每节规则独立成文件，按编号 1-18 对应。
- **`workflow.md`** — 使用工作流：何时调用、如何套用模板、如何校验、输出规范。
- **`templates/storyboard.template.md`** — 固定分镜模板（每镜统一格式）。

---

## 如何开始

### 步骤 1. 加载 Skill

> 「加载 seedance-storyboard-rules 技能。」

会读取 `SKILL.md` 入口，按「加载时必读」顺序加载各主题文件。

### 步骤 2. 提供场景素材

以下情况之一：

**A. 用户已有场景设定（人物关系、空间结构、光影时间）。**
发送设定 — 按第四节锁定空间、按第十四节统一光影后开始拆镜。

**B. 用户只有剧情意图。**
用一到两段话描述。先按第十三节建立场景空间结构，再按第四节锁定人物站位。

**C. 用户要求审查既有分镜。**
发送既有分镜 — 按 `workflow.md` 的「如何校验分镜是否符合规则」逐项检查。

### 步骤 3. 按工作流套用模板

标准循环（详见 `workflow.md` 的「如何套用分镜模板生成新分镜」）：

1. 第一步：空间与关系建立（参考第十三节、第四节、第十四节）。
2. 第二步：镜头拆分（参考第二节，A/B/C/D 镜拆分法，2-3s / 3-5s / 5-7s 时长建议）。
3. 第三步：填写分镜模板（使用 `templates/storyboard.template.md`）。
4. 第四步：表演与潜台词设计（参考第七、八、九节）。
5. 第五步：填写正向/负向提示词（以第一节通用质量基底为基础 + 选定风格提示词）。

### 步骤 4. 校验

按 `workflow.md` 的「如何校验分镜是否符合规则」清单逐项检查 16 个维度（空间连续性、焦点统一性、时长合理性、景别衔接、运镜服务性、表演具体性、潜台词设计、双层表演、群像多样、手机处理、物件叙事、场景稳定、光影统一、蒙太奇拆分、声音设计、模板完整）。

## 对技能的典型请求

| 请求 | 执行行为 |
|---|---|
| 「创作微短剧分镜」 | 读取 `workflow.md` → 按五步工作流套用模板 |
| 「拆分这个镜头」 | 参考第二节 A/B/C/D 镜拆分法与时长建议 |
| 「编写 Seedance 2.0 提示词」 | 以第一节默认提示词为基础 + 本镜具体描述 |
| 「审查既有分镜是否符合规则」 | 按 `workflow.md` 校验清单逐项检查 16 个维度 |
| 「设计电影级 CG 分镜」 | 参考第一节视觉渲染规则 + 第五节景别衔接 + 第六节运镜 |
| 「套用或修改固定分镜模板」 | 使用 `templates/storyboard.template.md` |
| 「设计双层人物表演」 | 参考第九节「表情在演，眼睛泄密」 |

## 输出规范

详见 `workflow.md` 的「输出规范」节。核心要求：

- 所有分镜输出必须使用 `templates/storyboard.template.md` 的固定模板。
- 正向提示词必须以第一节通用质量基底开头，后接选定风格提示词。
- 负向提示词必须包含：文字、字幕、水印、BGM、过曝、塑料皮肤。
- 连续分镜之间必须明确标注空间关系继承。
- 复杂段落必须显式标注 A/B/C/D 镜拆分。
