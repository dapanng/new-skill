# 固定优化输出模板

> 此模板用于对场景原提示词做光影/摄像机/氛围的正向视觉增强，使用时按需填写各项字段。约束：原提示词逐字保留，正向增强区不得出现否定句承担增强，末尾必须完成布局保真声明。

```text
【场景类型】室内 / 室外 · 具体子类型

原提示词（逐字保留）：

锁定锚点区：
- 人物：
- 物件：
- 空间结构：
- 动作意图：
- 时间天气（原提示词已指定 / 未指定已补充）：

正向增强区（光影）：

正向增强区（摄像机）：

正向增强区（氛围与色调）：

下游使用建议（可选：Seedance / HyperFrames / Remotion）：

布局保真声明：
- 已确认人物/物件/空间/动作与原提示词一致：
- 已确认正向增强区无否定句承担增强：

负向提示词（仅校验用，不计入正向视觉增强主体）：
```
