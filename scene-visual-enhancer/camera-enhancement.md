## 五、摄像机优化规则

机位增强只改变观察方式，不改变被观察的布局。

### 机位高度与角度

| 机位 | 画面效果（正向） | 情绪指向 |
|------|------------------|----------|
| 平视 | 呈现与人物平齐的客观视角与稳定构图 | 中立、叙事、平静 |
| 低角度仰拍 | 呈现人物高大感与天空/天花板背景 | 权威、压迫、英雄 |
| 高角度俯拍 | 呈现人物渺小感与环境关系 | 脆弱、审视、孤独 |
| 荷兰角 | 呈现场景倾斜的失衡透视 | 失序、紧张、心理动荡 |

### 焦距与景深

| 参数 | 画面效果（正向） | 适用场景 |
|------|------------------|----------|
| 广角 | 呈现夸张透视与空间纵深感 | 狭小室内、宏大场景、运动跟拍 |
| 标准 | 呈现接近人眼的自然透视与平稳构图 | 对话、日常叙事、人物中景 |
| 长焦 | 呈现空间压缩感与前后景贴近 | 隔离主体、远距离窥视、街头人物特写 |
| 浅景深 | 虚化前后景突出主体，呈现视觉聚焦 | 情绪特写、回忆段、关键物件 |
| 深景深 | 前后景均清晰，呈现环境关系与多层叙事 | 全景关系戏、空间交代、群像 |

### 运镜

| 运镜 | 正向描述 | 适用情绪 |
|------|----------|----------|
| 慢推 Slow Push-in | 镜头缓慢向主体推进，逐渐收紧视野至情绪落点 | 意识觉醒、情绪看透、心理压力增加 |
| 环绕 Arc Shot | 镜头绕主体约四分之一圈至半圈，呈现关系审视的视角变化 | 关系审视、人物被围困、权力关系变化 |
| 手持 Handheld | 镜头以手持节奏轻晃，服务于突发事件与慌乱氛围的临场感 | 舆论失控、人群骚动、慌乱、突发事件 |
| 静止 Static | 镜头静止凝视，呈现冷静客观的审视感 | 冷静审视、压迫、反讽、重要人物关系 |
| 焦点转移 Rack Focus | 焦点从动作人物转到反应人物，或从人物转到物件证据 | 动作-反应交接、人物-物件关联、前后景层次 |

运镜服务情绪，节奏与幅度服务于场景已有张力。

### 构图

- 三分法：主体置于三分线交点，呈现平衡与引导。
- 引导线：利用场景线条（走廊、栏杆、路面）引导视线至主体。
- 框中框：利用门窗、镜框、拱门在画面内构建二次框定。
- 前景遮挡：利用前景物件或人物虚化遮挡，呈现层次与窥视感。
- 对称构图：以中轴对称呈现仪式感与秩序感。

### 机位与布局的关系

- 原则：机位变化不得导致人物/物件的相对位置关系在画面中产生矛盾。
- 示例：原提示词「A 站在左、B 站在右」+ 机位「从 B 的右侧拍摄」→ 若该机位会让 A 在画面中出现在 B 的右侧，则与布局矛盾，应改选「从 A 的左侧拍摄」或「正面拍摄 A 与 B 的关系」。
- 处理：若某机位会导致布局矛盾，改选不矛盾的机位；不得为迁就机位而改写原布局。
